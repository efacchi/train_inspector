# train_inspector

`train_inspector` is a Python-based application designed for the inspection, monitoring, or analysis of train and railway transport data. But only for italian trains.

---

## ⚠️ Project Status

This project is currently a **Work in Progress (WIP)**. 
* **Note:** The application is in an early stage and may be unstable, unoptimized, or inefficient at the moment.
* **Current State:** However some baseline features are already functional and ready to use.

---

## 🚀 Key Features

* **Core Functionality:** Basic train data parsing and initial inspection logic are already working.
* **Real-time Monitoring:** Ugly, but works. Not efficient at all.

---

## 🛠️ Prerequisites

Make sure you have **Python 3.8+** installed on your Linux system.
